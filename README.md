# 股票數據客戶端

## 簡介

本專案包含兩部分：股票數據伺服器（Server）與股票數據客戶端（Client）。此專案為客戶端，負責接收伺服器端資訊並顯示這些資訊。

## 功能

### 客戶端

- 與伺服器建立連接以接收股票數據。
- 顯示實時股票數據。
- 提供用戶界面以調整數據更新頻率。

## 如何使用

### 客戶端

1. 在客戶端應用程式中輸入伺服器的IP地址和端口。
2. 點擊連接按鈕以接收數據。
3. 使用滑動條調整數據更新頻率。
